## フォー

ベトナムを代表する米麺のスープで、牛骨や鶏ガラから取った芳醇なだしが特徴の国民食。

### 材料（2人分）

- フォー（乾麺）：200g
- 牛薄切り肉：150g
- 牛骨：500g
- 玉ねぎ：1個
- しょうが：1片（30g）
- シナモンスティック：1本
- 八角：2個
- ナンプラー：大さじ2
- 砂糖：小さじ1
- 塩：小さじ1/2
- もやし：100g
- パクチー：適量
- ライム：1/2個
- 水：1.5L

### 作り方

1. 牛骨を沸騰した湯で5分茹でてアクを取り、水で洗う。玉ねぎとしょうがは皮ごと直火で表面を焼く。
2. 鍋に水1.5Lと牛骨、焼いた玉ねぎ・しょうが、シナモンスティック、八角を入れ、弱火で1時間半煮込んでスープを取る。
3. スープを濾し、ナンプラー、砂糖、塩で味を調える。
4. フォーを袋の表示通りに茹でて水気を切り、器に盛る。
5. 熱々のスープを注ぎ、薄切りの牛肉をのせてスープの熱で火を通す。もやし、パクチー、ライムを添えて完成。

### ポイント

- 玉ねぎとしょうがを直火で焼くことでスープに香ばしさと甘みが加わり、本場の味に近づく。
- 牛肉は極薄切りにし、熱いスープを注いで半生に仕上げるのがベトナム流。

## バインミー

フランス統治時代の影響を受けたベトナム風サンドイッチで、パリッとしたバゲットに具材をたっぷり挟んだ一品。

### 材料（2人分）

- ミニバゲット：2本
- 豚もも肉（焼き豚用）：200g
- レバーパテ：大さじ2
- 大根：100g
- にんじん：50g
- きゅうり：1/2本
- パクチー：適量
- 青唐辛子：1本
- 酢：大さじ2
- 砂糖：大さじ1
- ナンプラー：大さじ1
- マヨネーズ：大さじ1
- 塩：小さじ1/2

### 作り方

1. 大根とにんじんを細切りにし、酢、砂糖、塩を合わせた液に30分漬けてなますを作る。
2. 豚もも肉にナンプラーをまぶし、180度のオーブンで25分焼き、薄切りにする。
3. バゲットを横に切り込みを入れ、軽くトーストする。内側にレバーパテとマヨネーズを塗る。
4. 焼き豚、なます、きゅうりの薄切り、パクチー、輪切りの青唐辛子を挟んで完成。

### ポイント

- バゲットは外がパリッと中がふわっとした軽い食感のものを選ぶと本場の味わいに近づく。

## 生春巻き

ライスペーパーで新鮮な野菜やエビを巻いた、ベトナムの定番前菜。

### 材料（4本分）

- ライスペーパー：4枚
- むきエビ（茹で）：8尾
- ビーフン（乾麺）：50g
- サニーレタス：4枚
- にんじん：1/3本
- きゅうり：1/2本
- 大葉：8枚
- ミント：適量
- スイートチリソース：大さじ2
- ナンプラー：大さじ1
- ライム汁：大さじ1
- 砂糖：小さじ1
- にんにく（すりおろし）：少々

### 作り方

1. ビーフンを茹でて冷水で締め、水気を切る。にんじんときゅうりは細切りにする。
2. ライスペーパーをぬるま湯にさっとくぐらせ、まな板の上に広げる。
3. 手前にサニーレタス、ビーフン、にんじん、きゅうり、大葉、ミントを並べ、手前から一巻きする。
4. 両端を内側に折りたたみ、エビを断面が見えるよう並べてから最後まで巻く。
5. ナンプラー、ライム汁、砂糖、にんにくを混ぜたタレとスイートチリソースを添えて完成。

### ポイント

- ライスペーパーは浸しすぎるとべたつくので、硬さが少し残る程度にさっとくぐらせるのがコツ。
- エビは縦半分に切って断面を外側に向けると、仕上がりが美しくなる。

## ブンチャー

ハノイ名物の炭火焼き肉つけ麺で、甘酸っぱいつけだれに焼き肉と米麺を浸して食べる。

### 材料（2人分）

- ブン（米の丸麺、乾麺）：200g
- 豚バラ肉（薄切り）：150g
- 豚ひき肉：100g
- ナンプラー：大さじ3
- 砂糖：大さじ2
- 酢：大さじ2
- ライム汁：大さじ1
- にんにく（みじん切り）：2片分
- 水：150ml
- レタス：適量
- パクチー：適量
- ミント：適量

### 作り方

1. 豚バラ肉にナンプラー大さじ1とにんにく半量をまぶし、15分漬ける。豚ひき肉にも同様に下味をつけ、小さな団子状に丸める。
2. 豚バラ肉と肉団子を魚焼きグリルまたはフライパンで焼き色がつくまで焼く。
3. 水、ナンプラー大さじ2、砂糖、酢、ライム汁を合わせてつけだれを作る。
4. ブンを袋の表示通りに茹でて冷水で締め、水気を切る。
5. つけだれに焼き肉と肉団子を入れ、ブン、レタス、パクチー、ミントを添えて完成。

### ポイント

- つけだれは甘味・酸味・塩味のバランスが重要なので、味見しながら好みに調整する。

## コムタム

砕き米の上に甘辛い味付けの炭火焼きポークをのせた、サイゴンの庶民的なワンプレート料理。

### 材料（2人分）

- 砕き米（なければジャスミンライス）：2合
- 豚ロース肉（厚切り）：2枚（約300g）
- レモングラス（みじん切り）：1本分
- にんにく（すりおろし）：2片分
- ナンプラー：大さじ2
- 砂糖：大さじ1
- サラダ油：大さじ1
- 黒こしょう：少々
- きゅうり：1/2本
- トマト：1/2個
- 目玉焼き：2個
- ヌクチャム（つけだれ）：適量

### 作り方

1. 豚ロース肉に格子状の切り込みを入れ、レモングラス、にんにく、ナンプラー、砂糖、サラダ油、黒こしょうを合わせたタレに30分以上漬け込む。
2. 漬け込んだ肉を魚焼きグリルまたはフライパンで中火にし、両面にしっかり焼き色がつくまで焼く。
3. 米を炊き、皿に盛る。焼き上がった肉を食べやすく切って米の上にのせる。
4. きゅうりの薄切り、トマトのくし切り、目玉焼きを添え、ヌクチャムをかけて完成。

### ポイント

- 肉はしっかり漬け込むほど味が染みるので、時間があれば一晩冷蔵庫で寝かせると良い。
- レモングラスの香りが肝なので、できれば生のものを使う。

## バインセオ

ターメリックで黄色く色づけたパリパリの米粉クレープに、エビや豚肉、もやしを包んだベトナム風お好み焼き。

### 材料（2枚分）

- 米粉：100g
- ターメリックパウダー：小さじ1/2
- ココナッツミルク：100ml
- 水：100ml
- 塩：小さじ1/4
- むきエビ：100g
- 豚バラ肉（薄切り）：100g
- もやし：100g
- 玉ねぎ（薄切り）：1/4個
- サラダ油：大さじ3
- サニーレタス：適量
- ミント・パクチー：適量
- ヌクチャム（つけだれ）：適量

### 作り方

1. 米粉、ターメリック、ココナッツミルク、水、塩をボウルで混ぜ合わせ、生地を作る。30分ほど休ませる。
2. フライパンに油大さじ1を熱し、豚バラ肉と玉ねぎを炒め、エビを加えて火を通し、一度取り出す。
3. 同じフライパンに油大さじ1/2を足し、生地の半量を薄く広げて中火で焼く。
4. 生地の端がパリッとしてきたら、片側に炒めた具材ともやしをのせ、半分に折りたたむ。蓋をして2分蒸し焼きにする。
5. サニーレタスで包み、ミントやパクチーを添え、ヌクチャムにつけて食べる。

### ポイント

- 生地は薄く広げ、しっかり焼いてパリパリの食感に仕上げるのが美味しさの秘訣。

## カオラウ

ホイアン名物のもちもちした太麺を少量のスープと具材で和えて食べる、この街でしか味わえない特別な汁なし麺。

### 材料（2人分）

- カオラウ麺（なければ太めのうどん）：200g
- 豚バラ肉（かたまり）：200g
- 五香粉：小さじ1/2
- 醤油：大さじ2
- 砂糖：大さじ1
- ナンプラー：大さじ1
- にんにく（みじん切り）：1片分
- もやし：100g
- レタス：適量
- パクチー：適量
- ミント：適量
- 揚げワンタンの皮（またはせんべい）：適量
- 水：100ml

### 作り方

1. 豚バラ肉に五香粉をまぶし、醤油大さじ1、砂糖、にんにくと共に鍋に入れ、水をひたひたに加えて弱火で40分煮込む。
2. 煮込んだ豚肉を取り出して薄切りにする。煮汁を煮詰めてタレにする。
3. 麺を茹でて水気を切り、器に盛る。醤油大さじ1とナンプラーを合わせた少量のタレを麺に絡める。
4. 麺の上に豚肉、もやし、レタス、パクチー、ミントをのせ、煮汁をかける。
5. 砕いた揚げワンタンの皮をトッピングし、よく混ぜて食べる。

### ポイント

- カオラウは汁なし麺なのでスープは少量にとどめ、麺と具材をしっかり和えて食べるのが本来の食べ方。
- 揚げたワンタンの皮のサクサク感が食感のアクセントになるので、必ず添える。

## ボーコー

フランス料理のビーフシチューがベトナム風にアレンジされた、レモングラス香るスパイシーな牛肉煮込み。

### 材料（4人分）

- 牛すね肉（角切り）：500g
- にんじん：2本
- 玉ねぎ：1個
- トマト：2個
- レモングラス（叩いて潰す）：2本
- にんにく（みじん切り）：3片分
- しょうが（薄切り）：1片分
- トマトペースト：大さじ2
- カレー粉：大さじ1
- 五香粉：小さじ1/2
- ナンプラー：大さじ2
- 砂糖：大さじ1
- ココナッツミルク：200ml
- 水：500ml
- サラダ油：大さじ2
- バゲット：適量

### 作り方

1. 牛すね肉に塩こしょうをふる。鍋に油を熱し、肉の表面に焼き色をつけて取り出す。
2. 同じ鍋でにんにく、しょうが、玉ねぎを炒め、トマトペースト、カレー粉、五香粉を加えて香りを出す。
3. 牛肉を戻し、ざく切りのトマト、レモングラス、水を加えて沸騰させ、アクを取り、蓋をして弱火で1時間半煮込む。
4. 乱切りのにんじんとココナッツミルクを加え、さらに30分煮込む。
5. ナンプラーと砂糖で味を調え、バゲットを添えて完成。

### ポイント

- バゲットにスープを浸しながら食べるのがベトナム流。フランスパンとの相性が抜群。

## チェー

ベトナムの伝統的なスイーツで、豆や芋、寒天、フルーツなどを甘いココナッツミルクと合わせた温冷どちらでも楽しめるデザート。

### 材料（4人分）

- 緑豆（皮なし）：80g
- さつまいも：1本（200g）
- タピオカパール（小粒）：30g
- ココナッツミルク：400ml
- 砂糖：80g
- 塩：ひとつまみ
- パンダンリーフ（あれば）：1枚
- 水：300ml
- 砕いた氷：適量

### 作り方

1. 緑豆を水に1時間浸してから柔らかくなるまで茹で、軽く潰す。
2. さつまいもを1cm角に切り、柔らかくなるまで茹でる。タピオカパールも透明になるまで茹でて冷水に取る。
3. 鍋にココナッツミルク、砂糖、塩、パンダンリーフを入れて弱火で温め、砂糖が溶けたら火を止める。
4. グラスに緑豆、さつまいも、タピオカを層にして入れ、ココナッツミルクを注ぐ。砕いた氷をのせて冷たいチェーの完成。

### ポイント

- 具材を層状に盛り付けると見た目が美しくなる。食べる直前に全体を混ぜて楽しむ。

## バインクオン

蒸した薄い米粉の皮で豚ひき肉やきくらげを巻いた、ハノイの朝食の定番料理。

### 材料（2人分）

- 米粉：100g
- タピオカ粉：20g
- 水：250ml
- 塩：小さじ1/4
- サラダ油：小さじ1
- 豚ひき肉：150g
- きくらげ（乾燥）：5g
- エシャロット（みじん切り）：2個分
- ナンプラー：大さじ2
- 砂糖：大さじ1
- ライム汁：大さじ1
- 水（タレ用）：大さじ3
- にんにく（みじん切り）：1片分
- フライドオニオン：適量
- パクチー：適量

### 作り方

1. 米粉、タピオカ粉、水、塩、油を混ぜて生地を作り、30分休ませる。きくらげは水で戻してみじん切りにする。
2. フライパンでエシャロットを炒め、豚ひき肉ときくらげを加えて火を通し、ナンプラー少々で下味をつける。
3. 蒸し器に油を薄く塗った布を張り、生地を薄く流し入れて蓋をし、強火で1〜2分蒸す。薄い皮が出来たら布ごと取り出す。
4. 蒸し上がった皮に具材をのせて巻き、食べやすい大きさに切る。
5. ナンプラー、砂糖、ライム汁、水、にんにくを混ぜたタレをかけ、フライドオニオンとパクチーを散らして完成。

### ポイント

- 皮は極薄に仕上げるのがコツで、生地を流し入れたら素早く布の上で広げる。
- 蒸し器がない場合は、油を薄く塗ったフライパンに生地を薄く流して蓋をし、弱火で蒸し焼きにしても代用できる。
