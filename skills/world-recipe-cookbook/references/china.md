## 小籠包

薄い皮の中に肉汁たっぷりのスープを閉じ込めた上海名物の点心。

### 材料（4人分）

- 豚ひき肉：200g
- 鶏ガラスープのゼラチン固め：100g
- 長ねぎ（みじん切り）：1/2本
- 生姜（すりおろし）：小さじ1
- 醤油：大さじ1
- 紹興酒：大さじ1
- ごま油：小さじ1
- 塩：小さじ1/2
- 小籠包用の皮（薄めの餃子の皮で代用可）：24枚

### 作り方

1. 鶏ガラスープにゼラチンを溶かし、冷蔵庫で固めてから細かく刻む。
2. ボウルに豚ひき肉、長ねぎ、生姜、醤油、紹興酒、ごま油、塩を入れてよく練り混ぜる。
3. 刻んだゼラチンスープを肉だねに加え、崩さないようにさっくり混ぜ合わせる。
4. 皮の中央に肉だねをのせ、ひだを寄せながら包む。
5. クッキングシートを敷いた蒸し器に並べ、強火で8〜10分蒸す。

### ポイント

- ゼラチンスープが蒸すことで溶け出し、肉汁スープになる。肉だねに混ぜるときは潰さないよう注意する。
- 蒸し上がったら皮が破れやすいので、そっと取り出して黒酢と千切り生姜を添えていただく。

## 麻婆豆腐

花椒の痺れる辛さと豆板醤の旨味が効いた四川省を代表する豆腐料理。

### 材料（2人分）

- 絹ごし豆腐：1丁（300g）
- 豚ひき肉：100g
- 長ねぎ（みじん切り）：1/2本
- にんにく（みじん切り）：1片
- 生姜（みじん切り）：1片
- 豆板醤：大さじ1
- 甜麺醤：大さじ1
- 鶏ガラスープ：150ml
- 醤油：大さじ1
- 水溶き片栗粉：大さじ2
- 花椒（粉）：小さじ1
- サラダ油：大さじ2
- ごま油：小さじ1

### 作り方

1. 豆腐を2cm角に切り、塩少々を加えた湯で2分ほど下茹でして水気を切る。
2. フライパンにサラダ油を熱し、にんにく、生姜を香りが出るまで炒め、豚ひき肉を加えてパラパラになるまで炒める。
3. 豆板醤と甜麺醤を加えて炒め合わせ、鶏ガラスープと醤油を注ぎ入れる。
4. 豆腐をそっと加え、崩さないように2〜3分煮込む。
5. 水溶き片栗粉を回し入れてとろみをつけ、仕上げにごま油と花椒をふりかける。

### ポイント

- 豆腐を下茹ですることで崩れにくくなり、余分な水分も抜けて味が染みやすくなる。
- 花椒は食べる直前にふると香りと痺れが際立つ。

## 回鍋肉

一度茹でた豚バラ肉をキャベツと共に甘辛い味噌で炒めた四川の家庭料理。

### 材料（2人分）

- 豚バラ肉（薄切り）：200g
- キャベツ：1/4個
- 長ねぎ：1本
- ピーマン：2個
- にんにく（みじん切り）：1片
- 豆板醤：小さじ1
- 甜麺醤：大さじ2
- 醤油：大さじ1
- 砂糖：小さじ1
- 紹興酒：大さじ1
- サラダ油：大さじ1

### 作り方

1. キャベツは一口大にちぎり、ピーマンは乱切り、長ねぎは斜め薄切りにする。
2. 豚バラ肉を食べやすい大きさに切り、フライパンでカリッとするまで炒めて一度取り出す。
3. 同じフライパンにサラダ油とにんにくを入れて弱火で香りを出し、豆板醤を加えて炒める。
4. キャベツ、ピーマン、長ねぎを加えて強火でさっと炒め、豚肉を戻し入れる。
5. 甜麺醤、醤油、砂糖、紹興酒を合わせたタレを回しかけ、全体に絡めて仕上げる。

### ポイント

- 豚バラ肉は先にしっかり焼いて脂を出すことで、香ばしくカリッとした食感になる。
- キャベツは炒めすぎず、シャキシャキ感を残すのがコツ。

## 酸辣湯

酢の酸味とこしょうの辛味が絶妙に効いた、具だくさんの中華スープ。

### 材料（2人分）

- 絹ごし豆腐：1/2丁（150g）
- 卵：1個
- 豚バラ肉（薄切り）：50g
- たけのこ（水煮）：50g
- きくらげ（乾燥）：5g
- 長ねぎ（小口切り）：1/4本
- 鶏ガラスープ：500ml
- 醤油：大さじ1
- 黒酢：大さじ2
- こしょう：小さじ1/2
- ラー油：適量
- 水溶き片栗粉：大さじ2
- 塩：適量

### 作り方

1. きくらげを水で戻して細切りにし、たけのこと豚バラ肉も細切りにする。豆腐は細長く切る。
2. 鍋に鶏ガラスープを沸かし、豚肉、たけのこ、きくらげを加えて3分ほど煮る。
3. 豆腐を加え、醤油と塩で味を調える。
4. 水溶き片栗粉を回し入れてとろみをつけ、溶き卵を細く流し入れてふんわり固まるまで待つ。
5. 火を止めてから黒酢とこしょうを加え、器に盛りつけてラー油と長ねぎを散らす。

### ポイント

- 黒酢は火を止めてから加えることで、酸味と香りが飛ばずにしっかり残る。
- 卵はとろみをつけた後に入れるとふわっとした仕上がりになる。

## 北京ダック

パリパリに焼き上げたアヒルの皮を薄餅に包んで食べる北京の宮廷料理。

### 材料（4人分）

- 合鴨ロース肉（皮付き）：400g
- はちみつ：大さじ2
- 醤油：大さじ1
- 五香粉：小さじ1/2
- 紹興酒：大さじ1
- 薄餅（春巻きの皮で代用可）：12枚
- 長ねぎ（白髪ねぎ）：1本分
- きゅうり（千切り）：1本分
- 甜麺醤：適量

### 作り方

1. 合鴨ロースの皮面にフォークで数か所穴を開け、紹興酒と五香粉を全体にすり込んで30分おく。
2. はちみつと醤油を混ぜ合わせ、皮面にまんべんなく塗る。
3. 天板に網をのせて鴨肉を皮面を上にして置き、200度に予熱したオーブンで40〜50分焼く。途中15分おきにはちみつだれを塗り直す。
4. 焼き上がったら皮をそぎ切りにし、肉も薄くスライスする。
5. 薄餅に甜麺醤を塗り、鴨の皮と肉、白髪ねぎ、きゅうりをのせて巻いていただく。

### ポイント

- 皮面にはちみつだれを繰り返し塗ることで、飴色のパリパリした食感に仕上がる。
- 合鴨ロースは常温に戻してから焼くと、中まで均一に火が通る。

## 青椒肉絲

細切りの牛肉とピーマンをオイスターソースで手早く炒めた広東風の定番料理。

### 材料（2人分）

- 牛もも肉（薄切り）：200g
- ピーマン：4個
- たけのこ（水煮）：80g
- にんにく（みじん切り）：1片
- 醤油：大さじ1
- オイスターソース：大さじ1
- 紹興酒：大さじ1
- 砂糖：小さじ1/2
- 片栗粉：小さじ2
- サラダ油：大さじ2
- ごま油：小さじ1

### 作り方

1. 牛肉を細切りにし、醤油小さじ1、紹興酒小さじ1、片栗粉をもみ込んで10分おく。
2. ピーマンは種を除いて細切りに、たけのこも細切りにする。
3. フライパンにサラダ油大さじ1を強火で熱し、牛肉を広げて入れ、色が変わるまで手早く炒めて取り出す。
4. 残りのサラダ油とにんにくを入れ、ピーマンとたけのこを強火で1分ほど炒める。
5. 牛肉を戻し入れ、醤油、オイスターソース、紹興酒、砂糖を合わせた調味料を加えて手早く絡め、仕上げにごま油を回しかける。

### ポイント

- 牛肉に片栗粉をまぶしておくことで、炒めたときに柔らかく仕上がり、タレもよく絡む。
- 強火で短時間に仕上げることがシャキシャキのピーマンとジューシーな肉の秘訣。

## 水餃子

もちもちの厚めの皮で肉と野菜の餡を包み、茹でて食べる中国北方の主食。

### 材料（4人分・約30個）

- 強力粉：200g
- 薄力粉：100g
- 熱湯：160ml
- 塩：小さじ1/4
- 豚ひき肉：250g
- 白菜（みじん切り）：200g
- にら（小口切り）：1/2束
- 生姜（すりおろし）：小さじ1
- 醤油：大さじ1.5
- ごま油：大さじ1
- 塩：小さじ1/2
- こしょう：少々

### 作り方

1. 強力粉と薄力粉を合わせ、塩を加えて熱湯を注ぎ、箸でかき混ぜてからひとまとめにし、ラップをかけて30分休ませる。
2. 白菜のみじん切りに塩をふって10分おき、水気をしっかり絞る。豚ひき肉にすべての調味料と白菜、にらを加えて粘りが出るまでよく混ぜる。
3. 生地を棒状にのばして30等分に切り、丸めてから麺棒で直径8cmほどの円形にのばす。
4. 皮の中央に餡をのせ、半月形に包んでしっかり閉じる。
5. たっぷりの湯を沸かし、餃子を入れて浮き上がってから3〜4分茹でる。
6. 黒酢、醤油、ラー油を合わせたタレを添えていただく。

### ポイント

- 生地に熱湯を使うことで、もちもちとした弾力のある皮に仕上がる。
- 白菜の水気をしっかり絞ることで、餡が水っぽくならず皮が破れにくくなる。

## エビチリ

プリプリのエビを豆板醤とケチャップベースのチリソースで絡めた日本でも人気の一品。

### 材料（2人分）

- むきエビ：200g
- 長ねぎ（みじん切り）：1/2本
- にんにく（みじん切り）：1片
- 生姜（みじん切り）：1片
- 豆板醤：小さじ1
- ケチャップ：大さじ3
- 鶏ガラスープ：80ml
- 砂糖：大さじ1
- 酢：小さじ1
- 片栗粉：大さじ1
- 塩・こしょう：少々
- サラダ油：大さじ2
- 溶き卵：1個分
- 紹興酒：大さじ1

### 作り方

1. エビは背わたを取り、塩、こしょう、紹興酒で下味をつけ、片栗粉をまぶす。
2. フライパンにサラダ油大さじ1を熱し、溶き卵をふんわり炒めて取り出しておく。
3. 残りの油でにんにく、生姜を弱火で香りが出るまで炒め、豆板醤を加えて炒め合わせる。
4. エビを加えて中火で両面を焼き、ケチャップ、鶏ガラスープ、砂糖、酢を混ぜたソースを注ぐ。
5. ソースが絡んだら卵を戻し入れ、長ねぎを加えてさっと混ぜ合わせて仕上げる。

### ポイント

- エビに片栗粉をまぶしてから炒めることで、プリッとした食感になりソースもよく絡む。
- ふんわり卵を加えるとソースがまろやかになり、辛さの中にやさしい味わいが生まれる。

## 担々麺

芝麻醤のコクとラー油の辛味が重なり合う四川発祥の濃厚なスープ麺。

### 材料（2人分）

- 中華麺：2玉
- 豚ひき肉：100g
- 青梗菜：2株
- 長ねぎ（みじん切り）：1/2本
- にんにく（みじん切り）：1片
- 甜麺醤：大さじ1
- 芝麻醤（練りごま）：大さじ3
- 醤油：大さじ2
- 鶏ガラスープ：600ml
- ラー油：大さじ1
- 酢：小さじ1
- 砂糖：小さじ1
- 花椒（粉）：適量
- サラダ油：大さじ1

### 作り方

1. フライパンにサラダ油を熱し、にんにくを炒めて豚ひき肉を加え、パラパラになるまで炒めたら甜麺醤を加えて肉味噌を作る。
2. 丼に芝麻醤、醤油、酢、砂糖、ラー油を入れておく。
3. 鶏ガラスープを鍋で温め、丼の調味料に注いでよく混ぜ合わせてスープを作る。
4. たっぷりの湯で中華麺を茹で、同時に青梗菜もさっと茹でる。
5. 湯切りした麺をスープの丼に入れ、肉味噌と青梗菜を盛りつけ、花椒と長ねぎを散らす。

### ポイント

- 芝麻醤は温かいスープで溶くことで滑らかに混ざり、コクのあるスープに仕上がる。
- 肉味噌はしっかり水分を飛ばして炒めると、香ばしさが増して麺によく合う。

## 春巻き

パリパリに揚げた薄い皮の中に野菜と肉の具材がぎっしり詰まった広東風の点心。

### 材料（4人分・10本分）

- 春巻きの皮：10枚
- 豚バラ肉（薄切り）：100g
- 春雨（乾燥）：30g
- もやし：100g
- にら：1/2束
- にんじん：1/3本
- 干ししいたけ：3枚
- 醤油：大さじ1
- オイスターソース：大さじ1
- ごま油：小さじ1
- 塩・こしょう：少々
- 水溶き小麦粉（のり付け用）：適量
- 揚げ油：適量

### 作り方

1. 干ししいたけを水で戻して細切りにし、春雨を茹でて食べやすい長さに切る。豚肉、にんじんは細切り、にらは3cm幅に切る。
2. フライパンにごま油を熱し、豚肉を炒め、にんじん、しいたけ、もやし、にらの順に加えて手早く炒める。
3. 春雨を加え、醤油、オイスターソース、塩、こしょうで味付けし、バットに広げて冷ます。
4. 春巻きの皮を菱形に置き、手前に具をのせて左右を折り込みながら巻き、巻き終わりを水溶き小麦粉で留める。
5. 160度の油で3〜4分、きつね色になるまでじっくり揚げる。

### ポイント

- 具材は必ずしっかり冷ましてから包むこと。温かいまま包むと皮が湿気てパリッと揚がらない。
- 低めの温度からじっくり揚げることで、中まで火が通りつつ皮がパリパリに仕上がる。
